# Sistema-Salao-POO
Este sistema de gerenciamento de agendamentos para salão de beleza é um projeto de Arquitetura de Software que demonstra a correta atribuição de responsabilidades usando os Padrões GRASP. Seu objetivo é garantir um design de software orientado a objetos com Alta Coesão e Baixo Acoplamento, simulando a validação de conflitos de horário.
